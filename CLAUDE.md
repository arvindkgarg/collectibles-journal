# CLAUDE.md — collectibles-journal

Hugo static site for **inkandmint.com** — a collector's blog for rare books, stamps, coins & ephemera.

## Stack

- **Hugo** v0.163.3 + **Stack theme** v4.0.3
- Hosted on **GitHub Pages** (`arvindkgarg/collectibles-journal`, branch `main`)
- DNS via **Cloudflare** (A/CNAME records as DNS-only / grey cloud — required for GitHub Pages SSL)

## Common commands

```bash
cd /Users/arvindkg/Downloads/ws/claude/collectibles-journal

# Local dev server (default port 1313)
hugo server

# Production build
hugo --quiet

# Build + push to GitHub (triggers GitHub Pages deploy, ~1 min)
hugo --quiet && git add -A && git commit -m "..." && git push
```

## Blog post frontmatter

Every post in `content/posts/` follows this schema:

```yaml
---
title: 'Human-readable title'
slug: 'url-slug'
date: 2025-01-15T00:00:00Z
draft: false          # true = not live; false = published
categories: ['Books']
tags: ['tag1', 'tag2']
ebay_url: 'https://www.ebay.com/itm/...'
ebay_price: 85.0
cover:
  image: 'https://i.ebayimg.com/...'
  alt: 'Description'
---
```

Posts are generated by `blog_pipeline.py` in the marketplace-tools repo and written directly to this directory.

## Custom CSS (`assets/css/custom.css`)

Key overrides:

| Rule | Purpose |
|---|---|
| `.sidebar .site-name a` | Remove underline from SVG wordmark link |
| `.left-toc` | TOC widget in left sidebar (desktop only) |
| `@media (max-width: 767px) .article-page .right-sidebar { order: -2 }` | Mobile article layout: logo → widget → article |
| `.article-list--tile article:not(.has-image) .article-details` | Related content tile padding |
| `.article-content img` | Constrain inline post images (max 80% width, 520px height) |

## Mobile article layout (flex order)

On mobile (`< 768px`) for article pages, the flex order is:
1. Left sidebar (default `order: -3`) — logo + hamburger nav
2. Right sidebar (`order: -2`, ties with main but wins by DOM position) — "From the Collection" widget
3. Main content (`order: -2`) — article body

## Right sidebar widget (`similar-listings.html`)

Shows for every article page (configured in `hugo.toml` `[params.widgets]`):

1. **From the Collection** — cover image, price, "View on eBay →" button (`.ebay-btn`)
2. **Related posts** — up to 4 related articles (Stack theme `Related` pages)
3. **eBay search link** — "Search eBay for '{primary tag}' →"

## Publishing a new post

Posts are normally published via the marketplace-tools dashboard ("Save & Publish") or the blog pipeline (`--publish`). Both:
1. Flip `draft: true` → `draft: false` in the markdown
2. `git commit` + `git push`
3. Update `blog_posts.status` in crosslist.db to `published`

To manually publish:
```bash
# Edit the post: change draft: true → draft: false
git add content/posts/your-post-slug.md
git commit -m "Publish post: your-post-slug"
git push
```

## GitHub Actions / Deploy

Push to `main` triggers GitHub Pages deploy automatically (Actions workflow provided by GitHub Pages). Deploy takes ~60–90 seconds.

If a deploy fails: `gh run list` to find the run ID, then `gh run rerun <id>`.

## Theme customisation notes

- Stack theme is at `themes/stack/` — **never edit theme files directly**; always override via `layouts/` or `assets/css/custom.css`
- `{{ ... | safeJS }}` is required for JSON-LD inside `<script>` blocks — Go's html/template JS-escapes `"` inside script context if you use `safeHTML`
- The hamburger button (`#toggle-menu`) is `position: absolute` within the sidebar — on mobile it moves with the sidebar's flex order
